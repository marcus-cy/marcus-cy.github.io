### Python：一种交互式、解释性、面向对象的编程语言

解释性语言：即无需编译源码为可执行文件，直接使用源码就可以运行

#### Python shell

提供一个运行环境，方便交互式开发。在windows下，分为两种，Python(command line) 和 IDLE

#### Python IDE

将Python开发相关的各种工具集成，包括python代码编辑器，python运行环境（pyhton shell）。常见主流的IDE有Pycharm和spyder
